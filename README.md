# Holy Martyrs Faith, Hope, Love & Sophia Orthodox Church

A fundraising website for the Holy Martyrs Orthodox Church building fund in Chicago.

## About

This project is a single-page fundraising site for the Russian Orthodox Church Outside of Russia (ROCOR), Diocese of Chicago.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The site is deployed to GitHub Pages via GitHub Actions.
